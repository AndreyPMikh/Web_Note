# Описание проекта

Данный проект представляет собой веб-приложение, которое позволяет пользователям авторизоваться, зарегистрироваться, создавать, редактировать и удалять записи, а также просматривать историю операций. Вся информация хранится в базе данных, а API предоставляет серверные методы для реализации функциональности.

## Используемые языки и библиотеки

- Python 3.10+
- Flask
- SQLAlchemy > 2
- Flask-Login

## Установка и запуск

1. Склонируйте репозиторий:
	git clone https://github.com/AndreyPMikh/Web_Note/
2. Перейдите в директорию проекта:
	cd project-directory
3. Создайте и активируйте виртуальное окружение:
	python -m venv venv
	source venv/bin/activate
4. Установите зависимости:
	pip install -r requirements.txt
5. Создайте базу данных:
	flask db init
	flask db migrate -m "Initial migration"
	flask db upgrade
6. Запустите приложение:
	python main.py
